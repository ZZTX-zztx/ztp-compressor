# ZTP 加密压缩/解压器

私有加密格式 | 本地离线运行 | AES+XOR 双重加密 | 文件夹打包工具

## 项目介绍

ZTP 加密压缩/解压器是纯前端、单文件运行的加密打包工具。
使用自定义私有 `.ztp` 格式，普通解压软件无法识别，仅本工具可解密。

所有操作均在本地浏览器完成，不上传任何数据，安全、轻量、跨设备可用。

## 功能特性

- AES 密码加密 + XOR 双重加密
- 自定义私有 ZTP 文件格式
- 支持单文件、多文件、文件夹打包
- 支持点击选择与拖放文件
- 暗黑 / 亮色双主题
- 实时进度条显示
- 手机、电脑全平台支持
- 纯本地运行，无需安装

## 使用方法

### 加密打包
1. 打开 ztp-encryptor.html
2. 设置加密密码
3. 选择或拖入文件/文件夹
4. 点击生成 .ztp 加密包
5. 自动下载加密文件

### 解密解压
1. 输入加密时的密码
2. 选择 .ztp 文件
3. 点击解密并解压
4. 自动还原所有原始文件
